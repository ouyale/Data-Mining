# Kenya Job Boards Web Scraper

This Repo contains a Python web scraping tool to collect job descriptions from Kenyan job boards. The scraper fetches thousands of job listings, extracting details such as job title, company, location, salary range, posting date, and job description text.

## Features

- Scrapes multiple Kenyan job boards via the Firecrawl API
- Saves output as CSV and JSON files
- Configurable with environment variables (`env_template.sh`)
- Includes requirements file for easy setup
- Handles thousands of job descriptions (~5000+) for analysis
